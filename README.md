# pdf_to_chemdb

## О проекте

**pdf_to_chemdb** — это инструмент для автоматического извлечения информации о биологической активности химических соединений из научных статей в формате PDF. Скрипт парсит таблицы и текстовые абзацы, структурирует данные в единую таблицу (CSV), похожую на известные ресурсы **ChEMBL** и **BindingDB**. 

Проект позволяет быстро собрать базу данных по заданным мишеням (опиоидные рецепторы, CB1, NOP, NPSR и др.) без ручного ввода, что особенно полезно для SAR-анализа, машинного обучения и подготовки датасетов для QSAR-моделирования. Извлекаемые параметры включают Ki, IC50, pKi, pIC50, pKB, а также информацию о типе анализа (binding, functional) и условиях эксперимента.

**Особенности**:
- Извлекает таблицы с активностями (Ki, IC50, pKi, pIC50, pKB) из PDF-файлов.
- Распознаёт названия соединений и сопоставляет их с активностями.
- Автоматически определяет столбцы для разных мишеней по заголовкам (µ, δ, κ, ORL1, CB1, NPS и т.д.).
- Использует camelot и pdfplumber для извлечения таблиц, а также fallback-парсинг из текста.
- Экспортирует все данные в один CSV-файл `chem_data.csv` (85 записей), готовый для анализа.

## Установка и запуск

### 1. Установите зависимости
```bash
pip install -r requirements.txt
```
### 2. Подготовьте PDF-файлы
Поместите все PDF-файлы с научными статьями в папку `pdfs`. Скрипт обработает все файлы с расширением `.pdf`.

### 3. Запустите скрипт
```bash
python scraper.py
```

### 5. Получите результат
После выполнения в папке проекта появится файл chem_data.csv со всеми извлечёнными данными.

**Структура выходного файла (`chem_data.csv`)**

| Столбец            | Описание |
|--------------------|----------|
| `record_id`        | Уникальный идентификатор записи (UUID) |
| `source_pdf`       | Имя исходного PDF-файла |
| `source_doi`       | DOI статьи (если найден) |
| `source_page`      | Номер страницы |
| `source_table`     | Название таблицы |
| `compound_id`      | Идентификатор соединения (из статьи) |
| `endpoint_raw`     | Тип активности (`Ki`, `IC50`, `pKi`, `pIC50`, `pKB`) |
| `value_raw`        | Численное значение (в исходных единицах) |
| `unit_raw`         | Единицы измерения (`nM`, `pX` и т.д.) |
| `qualifier`        | Оператор сравнения (`>`, `<`, `=`) |
| `target_raw`       | Название мишени (например, `µ opioid receptor`) |
| `target_std`       | Стандартизованный идентификатор мишени (например, `OPRM1`) |
