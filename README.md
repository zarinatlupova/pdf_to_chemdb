# pdf_to_chemdb

## О проекте

**pdf_to_chemdb** — это инструмент для автоматического извлечения информации о биологической активности химических соединений из научных статей в формате PDF. Скрипт парсит таблицы и текстовые абзацы, структурирует данные в единую таблицу (CSV), похожую на известные ресурсы **ChEMBL** и **BindingDB**.

Проект разработан в рамках хакатона для демонстрации возможностей обработки научных данных и создания собственной химической базы данных. Он не привязан к конкретной мишени и может быть адаптирован для любых рецепторов, ферментов или белков.

**Особенности**:
- Извлекает таблицы с активностями (Ki, IC50, EC50, Ke и др.) из PDF-файлов.
- Распознаёт названия соединений и сопоставляет их с активностями.
- Автоматически определяет столбцы для разных мишеней по заголовкам (µ, δ, κ, ORL1, CB1, NPS и т.д.).
- Выделяет SAR-тренды из текста (необязательно, для демонстрации).
- Экспортирует все данные в **один** CSV-файл `chem_data.csv`, готовый для анализа.

## Установка и запуск

### 1. Установите зависимости
```bash
pip install -r requirements.txt
```
### 2. Подготовьте PDF-файлы
Поместите все PDF-файлы с научными статьями в папку `pdfs`. Скрипт обработает все файлы с расширением `.pdf`.

### 3. Запустите скрипт
```bash
python scraper.py
```

### 5. Получите результат
После выполнения в папке проекта появится файл chem_data.csv со всеми извлечёнными данными.

**Структура выходного файла**
| Столбец | Описание |
|---------|----------|
| `molecule_id` | Уникальный идентификатор (например, MOL001) |
| `name` | Название соединения (из статьи) |
| `smiles` | SMILES-строка (если доступна) |
| `clogp` | Коэффициент гидрофобности (если указан) |
| `target` | Название мишени (например, µ opioid receptor) |
| `assay_type` | Тип теста: `binding` (связывание) или `functional` (функциональный) |
| `activity_type` | Тип активности: `Ki`, `IC50`, `EC50`, `Ke` и др. |
| `value` | Численное значение (в nM) |
| `units` | Единицы измерения (по умолчанию `nM`) |
| `efficacy` | Эффективность (например, `% стимуляции`, `Emax`) |
| `selectivity_ratio` | Соотношение селективности (например, µ/κ = 150) |
| `source_pdf` | Имя исходного PDF-файла |
